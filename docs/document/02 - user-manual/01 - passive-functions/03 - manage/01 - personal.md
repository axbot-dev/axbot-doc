## 查看当前账号状态

查看用户当前账号的状态。返回的信息中**不包含敏感内容**，请放心在公共频道使用

| 一级命令   | 二级命令 | 返回内容介绍          | KOOK | QQ频道 | 备注 |
|--------|------|-----------------|------|------|----|
| 状态     | -    | 返回用户的一些状态数据，非敏感 | 支持   | 支持   | -  |
| 我的状态   | -    | 同上              | 同上   | 同上   | -  |
| 个人状态   | -    | 同上              | 同上   | 同上   | -  |
| status | -    | 同上              | 同上   | 同上   | -  |
