# 默认/无命令匹配

当未向AXBot提供任何命令，或者提供的命令匹配不到任何命令时，AXBot会返回一些默认的响应

## 未启用智能模式

当请求的用户未启用智能模式时，AXBot会返回固定的内容

| 一级命令     | 二级命令 | 返回内容介绍      | KOOK | QQ频道 | 备注 |
|----------|------|-------------|------|------|----| 
| -        | -    | 返回项目介绍      | 支持   | 支持   | -  |
| <任意错误命令> | -    | 返回项目介绍，错误命令 | 同上   | 同上   | -  |
