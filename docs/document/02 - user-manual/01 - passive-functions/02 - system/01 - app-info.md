# 应用信息

以下的功能均和AXBot本身的信息相关

## 获取帮助

给用户提供必要的帮助信息，方便用户快速获取和应用相关的资源

| 一级命令 | 二级命令 | 返回内容介绍                     | KOOK | QQ频道 | 备注                         |
| -------- | -------- | -------------------------------- | ---- | ------ | ---------------------------- |
| 帮助     | -        | 返回项目主页，用户手册等关键信息 | 支持 | 支持   | QQ频道中，暂无法提供任何链接 |
| 文档     | -        | 同上                             | 同上 | 同上   | 同上                         |
| help     | -        | 同上                             | 同上 | 同上   | 同上                         |

## 获取版本号

获取当前应用的版本号

| 一级命令 | 二级命令 | 返回内容介绍         | KOOK | QQ频道 | 备注 |
| -------- | -------- | -------------------- | ---- | ------ | ---- |
| 版本     | -        | 返回当前应用的版本号 | 支持 | 支持   | -    |
| version  | -        | 同上                 | 同上 | 同上   | -    |
